# 상태가 없는 React 컴포넌트

React 함수(재사용 용이, 클래스 보다 가벼움) 컴포넌트

- 상태(state)를 가지지 않음
- Prop에 의존하는 컴포넌트
- 표현적인(Presentational) 컴포넌트
- 마크업, 스타일링(레이아웃, 시각적 표현)

## 실습 컴포넌트

정규 수업 시간에 같이 만들 stateless 컴포넌트

- [ ] 인풋
- [x] 토글 버튼
- [x] 화면에서 감춰지지만, 접근성을 준수하는 컴포넌트
  - a11yHidden
  - readableHidden
  - srOnly
  - screenOff
  - blind
